Ejercicio 2. Contador de letras

Contar Vocales

Introduce una frase en el campo de texto y presiona el botón "Contar" para ver cuántas vocales (mayúsculas y minúsculas) hay en la frase. También averiguarás el número de letras de la frase.

Explicación

El código JavaScript cuenta las vocales en la frase ingresada por el usuario y muestra el resultado.

            
                document.getElementById('botonContar').addEventListener('click', contarVocales);
                document.getElementById('botonLimpiar').addEventListener('click', limpiarEntrada);

                function contarVocales() {
                    const frase = document.getElementById('fraseInput').value;

                    const contadorVocales = { 'a': 0, 'e': 0, 'i': 0, 'o': 0, 'u': 0, 'A': 0, 'E': 0, 'I': 0, 'O': 0, 'U': 0 };
                    let vocalesEncontradas = false;

                    let totalLetras = 0;
                    const letras = /[a-zA-Z]/;

                    for (let caracter of frase) {
                        if (contadorVocales.hasOwnProperty(caracter)) {
                            contadorVocales[caracter]++;
                            vocalesEncontradas = true;
                        }
                        if (letras.test(caracter)) {
                            totalLetras++;
                        }
                    }

                    if (!vocalesEncontradas) {
                        document.getElementById('textoResultado').innerHTML = "No se encontraron vocales.";
                    } else {
                        let textoResultado = '';
                        for (let [vocal, contador] of Object.entries(contadorVocales)) {
                            if (contador > 0) {
                                textoResultado += vocal + ' = ' + contador + '
';
                            }
                        }
                        document.getElementById('textoResultado').innerHTML = textoResultado;
                    }

                    document.getElementById('totalCaracteres').innerHTML = "Total de letras: " + totalLetras;
                    document.getElementById('cajaResultado').style.display = 'block';
                }

                function limpiarEntrada() {
                    document.getElementById('fraseInput').value = '';
                    document.getElementById('cajaResultado').style.display = 'none';
                }

Paso a paso:

  1. Agregar Event Listeners: Se añaden eventos a los botones de "Contar" y "Limpiar" para que llamen a las funciones correspondientes cuando se hace clic en ellos.
  2. Función contarVocales: Esta función se encarga de contar las vocales en la frase ingresada y el total de letras.
    • Obtiene el valor del input con id 'fraseInput'.
    • Inicializa un objeto 'contadorVocales' para contar cada vocal (tanto minúsculas como mayúsculas) y una variable 'totalLetras' para contar el total de letras.
    • Usa un bucle 'for...of' para recorrer cada carácter de la frase y, si es una vocal, incrementa su conteo en el objeto 'contadorVocales'. También incrementa 'totalLetras' si el carácter es una letra.
    • Si no se encontraron vocales, muestra un mensaje indicando esto.
    • Si se encontraron vocales, construye el texto del resultado solo con las vocales que tienen un conteo mayor a 0.
    • Muestra el total de letras en el elemento con id 'totalCaracteres'.
    • Muestra la caja de resultados estableciendo el estilo 'display' a 'block'.
  3. Función limpiarEntrada: Esta función limpia el valor del input y oculta la caja de resultados.
