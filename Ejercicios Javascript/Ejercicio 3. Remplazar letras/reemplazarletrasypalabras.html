Ejercicio 3. Reemplazar letras y palabras

Reemplazar letras y palabras

Introduce una frase en el campo de texto y presiona el botón "Reemplazar" para sustituir todas las letras 't', 'n', o 'p' por la palabra "curso". Presiona el botón "Revertir" para restaurar la frase original. Presiona "Limpiar" para borrar la entrada y el resultado.

Resultado:

Explicación

El código JavaScript reemplaza las letras en la frase ingresada y permite revertir los cambios.

            
                document.getElementById('botonReemplazar').addEventListener('click', reemplazarPalabras);
                document.getElementById('botonRevertir').addEventListener('click', revertirPalabras);
                document.getElementById('botonLimpiarReplace').addEventListener('click', limpiarEntradaReplace);

                let fraseOriginal = '';
                let fraseModificada = '';

                function reemplazarPalabras() {
                    const frase = document.getElementById('fraseInputReplace').value;
                    fraseOriginal = frase;
                    fraseModificada = frase.replace(/[tnp]/gi, 'curso');
                    document.getElementById('textoResultadoReplace').innerHTML = fraseModificada;
                    document.getElementById('cajaResultadoReplace').style.display = 'block';
                }

                function revertirPalabras() {
                    document.getElementById('textoResultadoReplace').innerHTML = fraseOriginal;
                }

                function limpiarEntradaReplace() {
                    document.getElementById('fraseInputReplace').value = '';
                    document.getElementById('cajaResultadoReplace').style.display = 'none';
                }

Paso a paso:

  1. Agregar Event Listeners: Se añaden eventos a los botones de "Reemplazar", "Revertir" y "Limpiar" para que llamen a las funciones correspondientes cuando se hace clic en ellos.
  2. Función reemplazarPalabras: Esta función reemplaza las letras 't', 'n', o 'p' por la palabra "curso".
    • Obtiene el valor del input con id 'fraseInputReplace'.
    • Reemplaza las letras 't', 'n', 'p' (mayúsculas y minúsculas) por la palabra "curso".
    • Muestra la frase modificada en el elemento con id 'textoResultadoReplace' y hace visible la caja de resultados.
  3. Función revertirPalabras: Esta función revierte las palabras "curso" a la frase original.
    • Muestra la frase original en el elemento con id 'textoResultadoReplace'.
  4. Función limpiarEntradaReplace: Esta función limpia el valor del input y oculta la caja de resultados.
